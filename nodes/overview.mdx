# ノードリファレンス

Noodlの主要な構成要素はノードです。各ノードにはそれぞれ特定の目的があり、単体では非常にシンプルですが、それらが合わさることで非常に強力になります。ここでは、すべてのコアノードとそのタイプ、カテゴリの概要を紹介します。

Noodlはモジュールで拡張することもでき、より多くのノードを使って構築できます。既に存在するモジュールについては、[モジュールライブラリ](/library/modules/overview)をチェックしてください。また、独自のモジュールを開発する方法については、[Noodl拡張セクション](/javascript/extending/overview)を参照してください。
