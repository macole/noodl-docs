# レシピアプリ

これは、[配列](/nodes/data/array/array-node)の操作方法を示すために作られたシンプルなレシピ管理アプリです。バックエンドには接続されておらず、[静的配列](/nodes/data/array/static-array)ノードを使用しています。**配列**からアイテムを追加・削除する方法や、いくつかのシンプルなフィルタリングを示しています。また、少しの**ナビゲーション**ロジックも含まれています。

どのように構築されているかを詳しく知るには、[配列ガイド](/docs/guides/data/arrays)をフォローしてください。
