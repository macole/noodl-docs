# タスクリストアプリ
これは、タスクリストアプリのシンプルな例です。Noodlの基本を学びたい場合に最適なスタートポイントです。**レイアウト**と**UIコントロール**を使用してシンプルなものを作成する方法を示しています。また、ユーザーによって作成されたタスクを保存するためのシンプルな**バックエンド**と**データベース**も使用しています。このアプリを完全に歩き通す[ビルドアロング](/docs/build-alongs/task-list-app)シリーズもあります。
