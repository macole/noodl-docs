# プリファブ

プリファブを使用すると、事前に構築されたコンポーネントをクローンしてプロジェクトで使用できます。コンポーネントはそのまま使用することも、開いて内部の構築ブロックを変更してニーズに合わせて調整することもできます。

プリファブをプロジェクトにインポートする方法については[こちら](/docs/guides/user-interfaces/modules)を参照してください。
