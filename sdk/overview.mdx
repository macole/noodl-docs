# SDK概要

Noodl SDK、または `@noodl/noodl-sdk` は、Noodlをコアノードで拡張しやすくするためのJavaScriptパッケージです。

## インストール

ここでは、SDKをインストールすることを含む、Noodlモジュールの作成方法についてのガイドがあります。
[こちらを続けてください。](/javascript/extending/overview)

## はじめに

私たちは、この[GitHubリポジトリ](https://github.com/noodlapp/modules)に多くのオープンソースモジュールを持っています。
これらは、モジュールの構築方法についての参照や例を探す際の素晴らしい出発点です。

これらのモジュールのすべてがSDKで作られているわけではありません。なぜでしょうか？
SDKは、内部Noodl APIの上に構築されており、内部Noodl APIの変更に対して将来的に互換性を持つように設計されています。
現在、SDKには内部Noodl APIが提供する機能の一部が不足しており、これがこれらのモジュールにSDKを使用しない主な理由です。
結局のところ、モジュールのコードは非常に似ていますが、混乱を招く可能性があります。

> この時点で、そのGitHubリポジトリにあるSDKを使用していない唯一のモジュールは、`simple-tooltips`モジュールです。

モジュールをNoodlアプリに登録するためには、SDKをインストールする必要はありません。
モジュールが登録される方法は、Noodlランタイムによってグローバルに定義された`Noodl.defineModule(module)`関数を呼び出すことにより、SDKではなく行われます。
