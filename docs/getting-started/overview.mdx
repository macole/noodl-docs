# Noodlの仕組み

Noodlには2つの基本的な部分があります：

1. **モダンなWebアプリケーションフロントエンドのためのビジュアルビルダー。** 高度にカスタマイズ可能なUIコントロールのライブラリからフロントエンドを構成します。細かなアニメーションやトランジションまでカスタマイズ可能です。シンプルなナビゲーションからより複雑なネストされたナビゲーションやポップアップをサポートする多機能なナビゲーションシステム。動的でリアクティブなインターフェースをサポートする再利用可能なコンポーネント。そして、ロジックを視覚的に、またはシンプルなJavaScript関数で構築します。

2. **統合されたクラウドサービス。** クラスとレコードを管理する簡単に学べる堅牢なデータベース。もちろん、データのクエリやレコードの関連付けも可能です。クラウドデータベースには、ユーザー管理（サインアップ、ログイン/ログアウト）とアクセスコントロールも含まれています。クライアント上でロジックを構築するのと同じように、クラウドで実行されるバックグラウンドジョブ、複合クエリ、セキュリティなどのためのクラウド関数を作成できます。これは、人気のあるオープンソースプロジェクト（[Parse](http://parseplatform.org)）に基づいており、完全にオープンソースで自己ホスト可能です。

## はじめに

Noodlの使い始めは簡単です。ツールに組み込まれたインタラクティブなレッスンが素晴らしい出発点です。ドキュメントでは、はじめに以下の記事を確認することをお勧めします。

- まず、[ワークフローの概要](workflow)ページとその後に[基本](fundamentals)ページを確認し、Noodlがあなたに何をもたらすことができるかについてもう少し詳しく知ることをお勧めします。

- 次に、[ガイド](/docs/learn)セクションに飛び込んで、Noodlの素晴らしい世界を掘り下げてみましょう。

また、[コミュニティチャンネル](https://www.noodl.net/community)もチェックしてみてください。学ぶのにも新しい友達を作るのにも素晴らしい方法です！

Happy Noodling！
