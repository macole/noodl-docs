# Noodlのナビゲーション

どんなアプリにも中心となるのがナビゲーションシステムです。Noodlは非常に柔軟なナビゲーションシステムを備えており、クラシックなWebスタイルのナビゲーション、アプリスタイルのナビゲーション、ポップアップを組み合わせて構築することができます。
